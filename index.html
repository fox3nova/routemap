系統更新日誌 / Changelog

v2.9.14 (Current) Latest

  • [修復] Safari / iOS 安全邊距與遮擋問題 (Safe Area / DVH Fix)

    將全域高度綁定從靜態的 100vh 升級為動態的 100dvh，並拉高地圖控制按鈕的底部安全邊距，徹底解決在 iPad Safari 中底部 UI 切斷或遮擋「重置按鈕」的問題。
    Upgraded viewport height to 100dvh and increased bottom safe-area padding for map controls to prevent overlap with Safari's dynamic navigation bar on iOS/iPadOS.

  • [修復] 核心資料庫遺失問題 (Core Database Reference Fix)
